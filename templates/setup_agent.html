✅ Device saved — install the agent

Because BBQ Guru's API blocks cloud servers, a small agent runs on your own computer and forwards data to your dashboard. It takes 2 minutes to set up.

🔑 Your Agent Token
{{ device.agent_token }}
1
Download the agent — save this file as cyberq_agent.py on your computer:
⬇ Download cyberq_agent.py
2
Run it in your terminal — copy and paste this command: 
CYBERQ_TOKEN="{{ device.agent_token }}" \
CYBERQ_USER="{{ device.fb_user }}" \
CYBERQ_PASS="{{ device.fb_pass }}" \
python3 cyberq_agent.py
3
Keep it running while you use the dashboard. The agent sends data every 12 seconds. You can run it in the background or in a separate terminal window.
💡 Tip: Want the agent to start automatically? On Mac, you can add it to Login Items or run it with nohup python3 cyberq_agent.py &.
